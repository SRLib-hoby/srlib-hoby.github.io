# Xiangyi Shaoting Art Action (Demo Collection)

This repository contains different frontend implementations for the "Xiangyi Shaoting Art Action" (相宜绍庭艺术行动) project. It serves as a showcase and comparison of different coding styles and AI-generated solutions.

## Available Implementations

### 1. Main Home
- **File**: [index.html](index.html)
- **Description**: The landing page acting as a directory for the demos.

### 2. Demo: Claude Implementation
- **File**: [index-claude-001.html](index-claude-001.html)
- **Tech Stack**: HTML, CSS (Custom Properties).
- **Style**: Custom design with "Cormorant Garamond" and "Outfit" fonts.

### 3. Demo: Gemini Implementation
- **File**: [index-gemini-001.html](index-gemini-001.html)
- **Tech Stack**: React (via CDN), Tailwind CSS, Babel (standalone).
- **Style**: Modern React functional components with Tailwind styling.

## How to Run

Since these are static HTML files, you can view them by:

1. **Locally**:
   - Clone the repository.
   - Open any `.html` file directly in your browser.

2. **Via GitHub Pages** (if enabled):
   - Visit `https://srlib-hoby.github.io/` (or the configured Pages URL).
